# Sort, filter and search in a react-app (Coding Challenge)

## with material-ui, jest, typescript, eslint, husky & lint-staged

###### Online Demo on Netlify: https://csb-7ujc7.netlify.app/

## Front-End Coding Challenge Definition

Link: [Reference to three API Requests and Definition of the Challenge (URL: Google Document) ](https://docs.google.com/document/d/1BXD-eZURPT36kGZyk_Zw5kRn92jtqNkFzg59RYABZPI/)

Welcome to this frontend challenge!

Your goal will be to build a web app that enables a user to search for shops, properties and agents in a unified list view. The unified list should be sortable by name and filterable by type.
